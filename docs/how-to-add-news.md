# お知らせの追加方法

このドキュメントでは、ウェブサイトにお知らせ（NEWS）を追加する方法を説明します。

## 手順

### 1. `lib/newsData.ts` を開く

お知らせのデータは `lib/newsData.ts` で管理しています。

### 2. 新しいお知らせを追加

`newsItems` 配列の**先頭**に新しいお知らせを追加します。

```typescript
export const newsItems: NewsItem[] = [
  // ↓ ここに新しいお知らせを追加
  {
    id: 9,  // 既存の最大ID + 1
    title: 'お知らせのタイトル',
    date: '2025.01.15',  // YYYY.MM.DD形式
    category: 'お知らせ',  // または 'サービス'
    slug: 'unique-slug',  // URLに使用される一意の識別子（英数字とハイフン）
    content: 'お知らせの本文をここに記載します。',
  },
  // 既存のお知らせ...
]
```

### 3. フィールドの説明

| フィールド | 説明 | 例 |
|-----------|------|-----|
| `id` | ユニークなID（既存の最大値+1） | `9` |
| `title` | お知らせのタイトル | `'新規案件受注のお知らせ'` |
| `date` | 日付（YYYY.MM.DD形式） | `'2025.01.15'` |
| `category` | カテゴリ | `'お知らせ'` or `'サービス'` |
| `slug` | URL識別子（英数字とハイフン、一意） | `'new-project-2025-01'` |
| `content` | 本文 | `'詳細な説明...'` |

### 4. 注意事項

- **slug は一意である必要があります**。既存のお知らせと重複しないようにしてください
- 新しいお知らせは配列の**先頭**に追加してください（新しい順に表示されます）
- 日付は `YYYY.MM.DD` 形式で記載してください

## 確認方法

1. 開発サーバーを起動: `npm run dev`
2. http://localhost:3000/news にアクセス
3. 追加したお知らせが一覧に表示されていることを確認
4. お知らせをクリックして詳細ページが正しく表示されることを確認
